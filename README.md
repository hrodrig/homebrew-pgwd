# homebrew-pgwd

Homebrew tap for [pgwd](https://github.com/hrodrig/pgwd) (Postgres Watch Dog) — a Go CLI that monitors PostgreSQL connection counts and notifies via Slack and/or Loki when thresholds are exceeded.

## Install

```bash
brew install hrodrig/pgwd/pgwd
```

## Links

- **Project:** [github.com/hrodrig/pgwd](https://github.com/hrodrig/pgwd)
- **Releases:** [pgwd releases](https://github.com/hrodrig/pgwd/releases)

The cask is updated automatically on each pgwd release (GoReleaser).
